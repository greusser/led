Editing commands

mouseleft
Draw
mouseright
Erase
alt + mouseleft
Pick existing value (in current layer)
alt shift + mouseleft
Pick existing value (in ANY layer)
space + mouseleft
Move the current level view
mouseWheel
Zoom
shift + mouseleft (held)
Draw a rectangle.
shift + mouseleft (single click)
Flood-fill the current area with the selected value.

Open editor panels
1 to 9
Select layers
ctrl S
Save project
ctrl shift S
Save project as...
ctrl W
Close project
ctrl Q
Quit app
G
Show/hide the grid
L
Toggle grid-locking (only works with some layer types)
R
Toggle auto-layers rendering
shift R
Toggle ALL auto-layers rendering
A
Toggle active layer enhancement
F
Fit current level in screen
H
Show this help
S
On TILE layers, save current tile selection.
R
On TILE layers, toggle randomization of current tile selection.
